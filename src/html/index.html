About Me

I'm on a mission to solve real problems at scale, particularly in healthcare.

To date, I've approached this in several ways including launching companies to tackle problems directly. Working as contract software engineer building websites, apps, and machine learning pipelines. And consulting for early-stage tech startups on product development, technical hiring, and outsourcing.

I'm currently completing my Master's thesis with Cambridge Cancer Genomics, a Y Combinator backed BioTech Startup. This is co-supervised by Prof. John Shawe-Taylor and applies Machine Learning to a particular bioinformatics problem.

Some of the things I've built include using machine learning to improve internet accessibility for those with Visual Impairments and developing a proof-of-concept for a faster, cheaper and more accurate test for macular degeneration. You can find more of my projects on Github.

I previously co-founded Proteam, which grew to be the largest platform for intra-university sport in the UK.

Prior to that, I studied Medicine at Cambridge. Where I was the founding President of the Cambridge StartUp Society which hosted events by startup founders and venture capitalists.
